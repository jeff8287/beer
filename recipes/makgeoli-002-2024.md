# 빚음 둘: 찹쌀 단양주 (민트 ver.)

> 단양주 · 양조일 2024-02-24

민트 넣은 버전과 안 넣은 버전 2종으로 나눠서 발효.

## Ingredients

| 재료 | 양 |
|------|----|
| 찹쌀 | 1.2 kg (1350 ml) |
| 금정산성 누룩 | 240 g |
| 양조수 | 720 ml |
| 와인 효모 | — |
| 민트 | (1버전만 추가) |

## Process

1. **쌀 불리기**: 찹쌀은 2시간 불림 (멥쌀은 3시간)
2. **고두밥**: 전기밥솥으로. 물 양은 쌀 부피의 **0.8배 = 1080 ml**
3. **누룩물**: 금정산성 누룩 240 g + 물 720 ml
4. 고두밥 + 누룩물 + 와인 효모 혼합
5. 두 버전으로 나눔 — **민트 추가 / 민트 없음**
